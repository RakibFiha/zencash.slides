# POSOS ZEN Manual

「POSOS」は株式会社Neukindが開発するPOSのためのノードです。
このマニュアルは、 2018 年 8 月末の情報に基づき株式会社Neukindが作成しております。ZenCashは非営利のZenブロッ
クチェーンファウンデーションによって管理されている仮想通貨です。株式会社 NeukindでZenCashの価格などについて保
証するものではありません。
